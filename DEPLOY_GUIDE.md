# iPhoneでWebアプリとして使うための手順

このアプリをiPhoneのホーム画面に追加して、アプリのように使うための手順です。

## 1. 準備（デプロイ）

iCloud Driveではなく、**Vercel** という無料のホスティングサービスを使うのが最も簡単で推奨されます。

### Vercelへのデプロイ手順

1.  GitHubアカウントをお持ちであれば、このプロジェクトをGitHubにプッシュし、Vercelと連携させるのが一番スムーズです。
2.  **GitHubを使わない場合（手動アップロード）**:
    *   ターミナルで `npm run build` を実行します（私が最後に実行しておきます）。
    *   `dist` というフォルダが作成されます。
    *   Vercelの公式サイト (vercel.com) に登録・ログインします。
    *   "Add New Project" -> "Upload" を選び、この `dist` フォルダをドラッグ＆ドロップするだけで公開できます。

## 2. iPhoneへのインストール

公開されたURL（例: `https://ped-pearls.vercel.app`）にiPhoneのSafariでアクセスします。

1.  画面下部の「共有」アイコン（四角から矢印が出ているアイコン）をタップします。
2.  メニューを下にスクロールして、**「ホーム画面に追加」** をタップします。
3.  右上の「追加」をタップします。

これでホーム画面にアイコンが追加され、タップするとアドレスバーのない全画面アプリとして起動します。
オフラインでも動作するように設定してあります（一度読み込めば、電波がなくても使えます）。
